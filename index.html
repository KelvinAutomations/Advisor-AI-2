Política de Privacidade — Advisor AI

Última atualização: Janeiro de 2026

O Advisor AI respeita a privacidade dos usuários e coleta apenas os dados estritamente necessários para cumprir seu único propósito: qualificar leads do LinkedIn e gerar recomendações comerciais acionáveis.

Dados coletados

  • Informações fornecidas pelo usuário no cadastro, como nome, sobrenome, profissão e e-mail.
  • Informações de autenticação, incluindo senha (tratada de forma segura no backend) e token de acesso (JWT).
  • Conteúdo público e visível de perfis do LinkedIn, coletado somente quando o usuário clicar em “Analisar”.

Dados que não coletamos

  • Dados de saúde
  • Dados financeiros ou de pagamento
  • Comunicações pessoais
  • Localização ou histórico de navegação
  • Monitoramento de cliques, digitação ou comportamento

Uso dos dados

Os dados são utilizados exclusivamente para autenticação, análise de perfis do LinkedIn e geração de score, temperatura do lead, abordagem, canal, produto e timing recomendados.

Compartilhamento

Não vendemos, não alugamos e não transferimos dados do usuário a terceiros fora do necessário para o funcionamento do serviço.

Segurança

Os dados são transmitidos utilizando criptografia moderna e tratados de forma segura.

Conformidade

O uso das informações segue a Chrome Web Store User Data Policy, incluindo os requisitos de Limited Use.
