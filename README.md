# Advisor-AI

Política de Privacidade — Advisor AI

O Advisor AI respeita a privacidade dos usuários e coleta apenas os dados estritamente necessários para cumprir seu único propósito: qualificar leads do LinkedIn e gerar recomendações comerciais acionáveis.

Dados coletados

Podemos coletar e processar:

Informações fornecidas pelo usuário no cadastro, como nome, sobrenome, profissão e e-mail.

Informações de autenticação, incluindo senha (tratada de forma segura no backend) e token de acesso (JWT).

Conteúdo público e visível de perfis do LinkedIn (ex.: nome, headline, resumo, experiências e educação), somente quando o usuário clicar explicitamente em “Analisar”.

Dados que NÃO coletamos

Não coletamos:

Dados de saúde

Dados financeiros ou de pagamento

Comunicações pessoais (e-mails, mensagens, chats)

Localização, IP ou coordenadas

Histórico de navegação

Monitoramento de cliques, digitação ou comportamento do usuário

Uso dos dados

Os dados são utilizados exclusivamente para:

Autenticação do usuário

Análise de perfil do LinkedIn

Geração de score, temperatura do lead, abordagem, canal, produto e timing recomendados

Compartilhamento de dados

Não vendemos dados do usuário.

Não compartilhamos dados com terceiros fora do estritamente necessário para o funcionamento do serviço.

Não utilizamos dados para publicidade, crédito, scoring financeiro ou fins de empréstimo.

Segurança

Os dados são transmitidos utilizando criptografia moderna e tratados de forma segura. Tokens de acesso são armazenados localmente apenas para manter a sessão do usuário.

Conformidade

O uso das informações recebidas segue integralmente a Chrome Web Store User Data Policy, incluindo os requisitos de Limited Use.
